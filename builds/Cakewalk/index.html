Overview

Grab a friend and collaborate to race to the finish! One player will control the red character while the other will control the blue one. Each character interacts with the environment differently, but characters have the ability to swap places to leverage their "strengths." There are a total of five exciting and challenging levels. Note: This game was made with a homemade Typescript engine, and as such will likely push the browser to the limit in an effort to hit 60 FPS. It is best not to have anything else constraining the CPU when you play.

Controls

It is recommended to play with gamepads. Plug two Xbox controllers in and P1 will control red and P2 will control blue. If you lack controllers, a keyboard will work as well.

Xbox gamepad controls
  • Left joystick: Run
  • A: Jump, and confirm menu selection
  • X: Swap
  • Start: Toggle menu
  • D-Pad: Browse menu options
Keyboard controls
  • Arrow Keys: P1 Run and Jump, and browse menu options
  • Enter (i.e. Return): P1 Swap, and confirm menu selection
  • WASD: P2 Run and Jump
  • Space: P2 Swap
  • Escape: Toggle menu
