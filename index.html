Hello and welcome! My name's Jay. I'm a software engineer interested in gaming, web and mobile development, image processing, build/deploy automation, and CS pedagogy — this list will likely grow longer as I learn more.

This site illustrates several projects I've worked on for classes, clubs, and just for fun. In some cases, portfolio items will be accompanied with builds that you can interact with through the site. These links are provided at the bottom of the page (please note that the builds are intended for desktop clients).

I've generated this site's webpages with a very basic templating system which interpolates YAML (for the portfolio) and Markdown (for the CV) into HTML snippets using short Python scripts. You are welcome to adapt this system for your own purposes: check out the repo. The front-end stack is Bootstrap 4 + jQuery + Font Awesome.
